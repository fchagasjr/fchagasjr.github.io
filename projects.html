LOGS BUDDY

Built on Sinatra, this app efficiently manages equipment maintenance. Users create logs, adding assemblies and individual equipment entities. It supports maintenance requests and records service data. Active Records and PostgreSQL ensure robust database functionality, enhancing data security. User authentication and session features to control access. Logs are shareable among other platform users via access keys with different permission levels.

Technologies: Sinatra, Active Record, PostgreSQL

Link: http://www.logsbuddy.com
